# Projet Unity VR - Mini-Golf de Bureau
**Étudiant :** Morhag Idriss et Hammadi Nour-eddine
**Date :** Février 2026

## Description
Ce projet est une application de VR simple mais fonctionnelle réalisée sous Unity 6. Il s'agit d'un stand de mini-golf où le joueur doit utiliser un club pour envoyer une balle sur une cible.

## Fonctionnalités validées
* **Interaction VR :** Saisie physique du club et de la balle.
* **Physique :** Utilisation du "Velocity Tracking" pour permettre une frappe réaliste du club sur la balle.
* **Locomotion :** Système de téléportation (Teleportation Area) pour se déplacer autour de la table.
* **Interface :** Canvas en "World Space" affichant le score en temps réel.

## Installation
1. Télécharger le fichier `Hammadi_projetV1.apk` présent dans le dossier `/Builds`.
2. L'installer sur un Meta Quest via SideQuest ou via `adb install`.
